# Additive Manufacturing References

## Scope Of This Section

This page is for representative papers on AI methods applied to additive manufacturing processes such as `LPBF`, `LDED`, `DED`, and `WAAM`.

The emphasis should remain on manufacturing-relevant use rather than on generic machine learning techniques with only superficial process mention.

## Suggested Subcategories

- process monitoring
- defect detection
- quality prediction
- process modelling
- parameter optimisation
- closed-loop control
- reinforcement learning
- digital twin integration

## How Papers Should Be Recorded

Each entry should capture:

- manufacturing process
- problem addressed
- AI method family
- sensing or input modality
- output target
- validation setting
- why the paper is representative
