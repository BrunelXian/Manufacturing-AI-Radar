## Scope Of This Section

This page is for papers that model manufacturing process behavior, state evolution, material response, or process-to-quality relationships using data-driven or hybrid approaches.

- hybrid physics-ML models
- uncertainty-aware modelling
- process response modelling
- material-property prediction
- state transition modelling
