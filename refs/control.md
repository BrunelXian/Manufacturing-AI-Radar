# Closed-loop Control References

## Scope Of This Section

This page is for papers on adaptive manufacturing control, learning-enhanced control, and feedback systems that use sensed or inferred process information to adjust actions.

## Suggested Subcategories

- adaptive control
- model predictive control
- learning-based control
- controller tuning
- constrained control
- quality-aware feedback control

## How Papers Should Be Recorded

Entries should capture the control loop structure, action variables, latency assumptions, and evidence of practical feasibility.
