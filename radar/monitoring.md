# Process Monitoring

## What This Domain Covers

This domain covers AI-enabled observation and interpretation of manufacturing processes using real-time or near-real-time data streams.

It includes monitoring based on images, thermal fields, acoustic signals, vibration, force, electrical signals, and multimodal industrial sensing.

## Why It Matters In Manufacturing

Manufacturing quality and stability are often determined during the process, not only at the final inspection stage. Monitoring provides the operational visibility needed to detect deviations early, support quality assurance, and create the information base for prediction and control.

## Typical Tasks

- process state estimation
- anomaly detection
- drift detection
- online quality indication
- machine health monitoring
- multimodal sensor fusion

## Representative Methods

- computer vision for in-situ observation
- time-series learning for industrial sensor streams
- anomaly detection for deviation identification
- multimodal fusion models
- representation learning for process signatures
- state estimation methods combining physics and data

## Example Application Scenarios

- thermal monitoring of additive manufacturing builds
- vibration-based condition monitoring in machining
- acoustic monitoring of tool wear or fracture events
- weld pool observation for welding stability
- process alarm systems based on fused sensor channels

## Key Research Questions

- which sensing combinations provide the highest practical value
- how should monitoring models handle noisy and partially observed industrial data
- how can monitoring outputs be made trustworthy for operators
- how transferable are monitoring models across machines and process setups
- what level of latency is acceptable for deployment into control loops

## Future Opportunity

Future opportunity lies in robust multimodal monitoring systems that provide interpretable state information for prediction, optimisation, and control.
