# Manufacturing-AI-Radar
A curated research radar exploring the impact of artificial intelligence on manufacturing systems, designed for learners, researchers, and industry practitioners.


当时想做的 Manufacturing-AI-Radar，本质上是一个“工业 AI 技术情报雷达 + 知识导航站”。
目标不是做论文数据库，而是 持续扫描、分类、解释 AI 在制造业中的应用趋势，类似一个 AI for Manufacturing 的技术地图（Technology Radar）。

换句话说：

不是存论文，而是建立“工业 AI 技术版图”。

这和普通 GitHub “awesome list” 是不同定位。
