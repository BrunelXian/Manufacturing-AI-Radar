# Notes

This directory is reserved for exploratory notes, synthesis memos, and early-stage topic fragments.

Use `notes/` for material that is promising but not yet ready for the main radar or stable documentation.

Preferred note types:

- literature scan summaries
- comparison notes across methods
- emerging topic memos
- open question lists

Reusable templates:

- [paper-note-template.md](paper-note-template.md)
- [domain-scan-template.md](domain-scan-template.md)
- [weekly-scan-template.md](weekly-scan-template.md)

Use these templates to convert broad scan output into structured research notes before promoting material into `refs/` or `radar/`.
